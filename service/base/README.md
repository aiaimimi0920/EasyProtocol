# EasyProtocol

`EasyProtocol` is the unified outward-facing service inside `ProtocolService`.

Current focus at the skeleton level:

- stable external service boundary
- unified protocol access entrypoint
- delegation or routing toward language-specific protocol services
- shared configuration and service-facing normalization
- strategy mode
- specified mode
- error cooling
- error attribution
- unified API interface

No executable implementation has been added yet.
