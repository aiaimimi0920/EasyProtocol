# Milestones

## Milestone 1: Bootstrap Baseline

Criteria:

- target repo exists
- root docs and migration plan exist
- source analysis and task plan exist

## Milestone 2: Structural Import Complete

Criteria:

- gateway and provider source trees are imported into monorepo-native paths
- deploy assets are imported into mirrored `deploy/` paths
- excluded runtime artifacts are removed from the target repo

## Milestone 3: Public Monorepo Operator Model

Criteria:

- root `config.example.yaml` exists
- derived config renderers exist
- contributors no longer need to edit imported per-module runtime configs

## Milestone 4: Hosted Automation Baseline

Criteria:

- validation workflow exists
- GHCR publish workflow(s) exist
- CI can materialize operator config from granular secrets

## Milestone 5: Release-Ready Public Repo

Criteria:

- root build / deploy / release scripts exist
- verification entrypoint exists
- public docs cover quickstart, config, secrets, and release flow
